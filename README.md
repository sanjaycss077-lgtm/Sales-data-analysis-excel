# Sales Data Analysis using Excel

This project focuses on analyzing sales data using Microsoft Excel to generate insights and support decision making.

## 🚀 Features
- Data cleaning and preprocessing
- Use of Pivot Tables for analysis
- Creation of charts and dashboard
- Identification of sales trends and KPIs

## 🛠 Tools Used
- Microsoft Excel

## 📊 Outcome
Analyzed sales performance, identified trends, and created a dashboard for better visualization of data.
